Hide the Website / URL input box on the comment form, supports Genesis 2.X, Thesis 2.X and some other WordPress themes (may not work on all themes)

== Description ==

Simply active this plugin to remove the website / url box from the WordPress comment form area.

This plugin has been tested on:

*   Genesis 2.X
*   Thesis 2.X
*   WordPress Twenty Fourteen theme

While it may work on other themes, there is no guarantee. If it doesn't work on your theme let me know in the support forums.

== Installation ==

1. Upload the folder `remove-comment-url-box` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The website URL box should now be gone, if not see the FAQ for this plugin.

== Frequently Asked Questions ==

= Why did this plugin not work? =

This plugin may not work with every WordPress theme / framework out there. Drop us a note if it doesn't work with your theme and let us know.

= Where are the options for this plugin? =

There are none, all you need to do is activate the plugin, the rest is done for you.
